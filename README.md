 *Overview*

This project is an AI-powered Attendance System that uses Computer Vision and Face Recognition to automatically mark attendance.
It detects faces in real-time using a camera, matches them with registered student data, and updates attendance records instantly.

**Features**

✅ Real-time face detection using webcam
✅ Face recognition for registered users
✅ Automatic attendance marking
✅ Stores attendance in CSV / Database
✅ Prevents duplicate attendance (same person cannot mark twice)
✅ Easy to register new students/users
